# Data Structure and Algorithms

Example C programs based on this book, for learning purposes.

### Attila Házy, Ferenc Nagy - Adatstruktúrák és algoritmusok

To build all examples, simply `git clone`, then `make`.
The resulting binaries will be in the `bin/` folder.

> The project is built on MacOS, but should work on any kind of Linux with C-compiler installed, or under MSYS2 / WSL on Windows.
